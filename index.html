Ken Burns Slideshow

A simple, lightweight, and responsive jQuery slideshow by Kevin Thornbloom that features Ken Burns type animations.

Features

  • Lightweight [3.5Kb]
  • Four Animation Types: Zoom In, Zoom Out, Pan Left, Pan Right
  • Easily add captions
  • CSS3 transitions for smoother effects
  • Responsive image sizes (resize this page)

Usage

  1. Include jQuery, kenburns.slides.js, and kenburns.styles.css 
    <link rel="stylesheet" href="css/kenburns.styles.css">
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.js"></script>
<script type="text/javascript" src="js/kenburns.slides.js"></script>
  2. Create a div with a class of "kb-slides". Create each slide within as a div called "kb-slide" and place your image inside. Images should be the same size. 
    <div class="kb-slides">
    	    <div class="kb-slide">
    	        <img src="images/1.jpg" />
    	    </div>
    	    <div class="kb-slide">
    	        <img src="images/2.jpg" />
    	    </div>
    	    <div class="kb-slide">
    	        <img src="images/3.jpg" />
    	    </div>
    	    <div class="kb-slide">
    	        <img src="images/4.jpg" />
    	    </div>
    </div>
  3. Want a caption? Add a title attribute to your "kb-slide" div. 
    <div class="kb-slide" title="Who brought the smores?">
	<img src="images/3.jpg" />
</div>

Customization

The slideshow will randomly pick animation effects as it plays. The timing of the slideshow is set in this line within the javascript file: 
var playTimer = 4000
This is in milliseconds. The styling of the slideshow can be changed by editing the kenburns.styles.css file.
